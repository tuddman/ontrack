## Ontrack ##

A mobile app to help college students get to Graduation.


setup:
```bash
$ sudo gem install xcodeproj
```

run (development):

```bash
$ sudo npm install -g ionic
$ sudo npm install -g ios-sim
$ cd ontrack
$ ionic serve
```
